# IDEA

    俗话说的好，工具善其事，必先利其器。
    在日常的后端开发过程中，通常开发工具逐步趋向于使用InteliJ IDEA。在工作中，时常会发现很多同事基于IDEA的操作不科学导致实现上的复杂，或害怕从传统的ECLIPSE转到IDEA。殊不知，如果在开发过程中掌握了几个必要的快捷键或者实现方式，便会让开发人员在开发过程中事半功倍，达到意想不到的效果。

环境准备

  • git
  • jdk1.8:springboot推荐jdk1.7以上
  • maven 3.x
  • IDEA(旗舰版/社区版均可)下载地址
