# IDEA插件

IDEA插件安装步骤:File->Setting->Plugins->Marketplace->搜索需要安装的插件。安装后File->Setting->Other settings可以对有些安装的插件进行简单的设置。

  • lombok 代码注解插件 添加注解 @Data，即可省去手写getter, setter, toString ,constructor的麻烦。具体使用请查看lombok在springboot中的使用

  • Spring Assistant spring项目助手在使用IDEA社区版时没有spring的官方快捷设置，这个时一个帮助快速初始化spring项目的插件工具。

  • Alibaba Java Coding Guidelines Alibaba开发手册的标准编码指导java开发手册（嵩山版）

  • Translation 翻译插件，选中后右键悬着Translate/Translate and Replace。

  • Key Promoter X 快捷键提示插件，每当操作右下角就会有快捷键提示，在实战中不断学习。get到后，可点击Don't show again不再提示。

  • BackgroundImage Plus + IDEA背景太单调，这是一个不错的选择，支持文件夹轮询播放. File->Settings->Appearance & Behavior->Background Image Plus->

  1. 设置Backgrounds背景图片文件夹
  2. 设置Change every多少时间和时间单位切换
  3. 设置Change mode切换模式Both/Keep same image保持同一张图片

  • Rainbow Brackets 同括号相同颜色，不同括号彩色括号。

  • Grep Console 高亮log不同级别日志，看日志的时候一目了然。

  • .ignore 支持自动创建gitignore文件，会标明项目中被忽略的文件，菜单选择需要忽略的文件/右键加入gitignore文件中

  • String Manipulation 字符串工具 选中文本后右键->

  1. 文本转换
  2. 样式转换
  3. 编码转换
  4. 递增/递减
  5. 排序
  6. 过滤/删除/移除
  7. 对齐
  8. 交换字符/选择/线/标记
  9. 引用间转换"" '' ``
  10. 切换文件路径分隔符：Windows <-> UNIX
  • Maven Helper maven依赖管理工具 pom文件打开有->Dependency Analyser->
  1. 查看依赖冲突/解决冲突
  2. 查看依赖树

  • Mybatis log plugin sql日志转换插件 选中执行sql和参数 右键Restore Sql From Selection

  • Code Search 在Baidu/Google/StackOverflow/GitHub搜索代码

  • GsonFormat JSON转领域对象工具 创建一个空类->选中类名右键->generate->GsonFormat(Alt+s)->填入json格式字符串->字段属性做调整后确定

  • Database Navigator 数据库连接插件支持Oracle/MySQL/SQLite/PostgreSQL/旗舰版IDEA自带Database菜单 连接测试注意安装插件/

连接好数据库，选择表->mybatis-generator->可创建数据库相关的类、dao层接口、xml文件

  • Codota 辅助开发神器 代码提示(list做示范)/类或代码选中右键->Ger relevant examples->查寻其他的可以add添加(ThreadPoolExecutor做示范)

  • Free Mybatis plugin 安装此插件后可以节约很多的开发时间, 在dao层接口可以直接进入xml文件中 点击dao层的mapper接口文件左边有绿色箭头/xml文件左边绿色箭头

TIP

  1. IDEA是非常消耗内存的IDE,所以对于日常没有用到或者不了解的插件尽可能不要选择，有助于减少内存消耗，以及提高启动该速度。
  2. 如果像更改或者发现插件运行错误，可以通过File->Setting->Plugins->Installed来进行启用或禁止。
